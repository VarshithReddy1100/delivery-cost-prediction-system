# Real-Time Delivery Cost Prediction System

This project is an end-to-end machine learning application designed to predict delivery costs in real time. It uses features such as distance, package weight, delivery type, traffic conditions, weather, and location-based inputs to estimate the final delivery price.

The project covers the complete ML workflow, including data generation, data preprocessing, model training, evaluation, backend API development, frontend integration, Docker containerization, and deployment.

## Key Features

- Real-time delivery cost prediction
- Machine learning model training and evaluation
- Backend API for prediction requests
- User-friendly frontend interface
- Dockerized application for easy deployment
- End-to-end project structure suitable for production-level ML deployment
